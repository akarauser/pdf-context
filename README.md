# pdf-context
PDF Context Retrieval

This script provides a context retrieval system for answering questions based on uploaded PDF documents. It utilizes PDF extraction, text splitting, vector embeddings, and a Langchain chain to process uploaded PDFs and respond to user queries. The system leverages FAISS for efficient similarity searches and Ollama for LLM interaction. The core functionality focuses on accurate information retrieval, making it a versatile tool for knowledge-based applications.
